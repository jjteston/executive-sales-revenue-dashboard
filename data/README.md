# Workbook Documentation

The workbook `Superstore_Analysis.xlsx` contains all spreadsheet-based work completed for this project.

## Worksheets

### Original Dataset

Contains the raw Superstore dataset used for analysis.

### Cleaned Dataset

Contains the validated dataset used for KPI calculations, pivot analysis, and dashboard development.

### Data Validation

Documents validation checks performed on the dataset, including missing values, duplicate records, negative-profit transactions, high-discount transactions, and invalid sales values.

### KPI Calculations

Contains the business KPIs defined for the project, including Total Revenue, Total Profit, Profit Margin, and Total Orders.

### Pivot Analysis

Contains exploratory analysis used to identify trends, regional performance, category performance, profitability drivers, and discount impacts.

### Reference Table

Contains a lookup table used to demonstrate XLOOKUP and data enrichment techniques.

### Business Requirements

Maps business questions to metrics and dashboard visualizations.
